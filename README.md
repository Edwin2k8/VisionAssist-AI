# 👁️ VisionAssist AI

VisionAssist AI is a Computer Vision application built using Python, OpenCV, YOLOv8, Streamlit, and SQLite.

## Features

* 🎥 Live webcam feed in Streamlit
* 🤖 Real-time object detection using YOLOv8
* 👤 Face detection and face counting
* 📊 Analytics dashboard with charts
* 🗄️ SQLite database for detection history
* 📥 CSV export functionality

## Technologies Used

* Python
* OpenCV
* YOLOv8
* Streamlit
* SQLite
* Pandas
* Matplotlib
* Streamlit WebRTC

## Dashboard Pages

* 🏠 Dashboard
* 📹 Live Camera
* 📋 Detection History
* 📊 Analytics
* ℹ️ About

## Installation

```bash
pip install -r requirements.txt
streamlit run dashboard.py
```
